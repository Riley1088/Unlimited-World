Machines Guide

Breath Amplification Machine (Map 1)

Boosts Attack Power using Amplification Gems.

  • Desert: +100% (56%)
  • Sea: +200% (30%)
  • Love: +300% (10%)
  • One Piece: +500% (3%)
  • Immortal Birds: +700% (0.8%)
  • Haoushoku Haki: +1000% (0.19%)
  • Gear 5: +2000% (0.01%)

How to get: Buggy (Map 1), Tower, Countdown Boss, Superboss, Pumpkin Chest

Talisman Machine (Map 4)

Boosts Skill Damage using Back Ornament Cards.

  • Wooden Box: +50% (60%)
  • Hulu: +100% (30%)
  • Pochita: +200% (8.9%)
  • BW Bears: +400% (1%)
  • Butterfly Wings: +1000% (0.1%)

How to get: Sasuke [Curse] (Map 4), Tower, Countdown Boss, Superboss, Pumpkin Chest

Awakening Machine (Map 8)

Boosts DMG using Awakening Potions.

  • Stripe Yorichi: +25% (60%)
  • Stripe Genya: +50% (30%)
  • Stripe Muichirou: +100% (8.9%)
  • Stripe Giyuu: +200% (1%)
  • Stripe Tanjiro: +300% (0.1%)

How to get: Inosuke (Map 8), Tower, Pumpkin Chest, Countdown Boss, Superboss
